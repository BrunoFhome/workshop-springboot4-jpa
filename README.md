# Sobre o projeto

Este projeto foi um Workshop realizado no curso de Java Completo do Professor Nelio Alves de DevSuperior.


A aplicação consiste em um sistema similar a um E-Commerce, aonde existem usuarios, pedidos, itens, entre outras entidades. Utilizando JPA/Hibernate para fazer a comunicação de todas as classes com o banco relacional.

# Tecnologias utilizadas
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
